# Usage
Add `call cl.bat` at the beginning of your batch script.

```batchfile
@echo off
title My first program
call cl.bat
echo %fg`red%Hello World!%`r%
pause
```

Every string must be closed with ```%`r%```.

# cl code explanation
Example: ```%fg`cyan-%```

Code | Explanation
---: | :---
```fg``` | foreground
```cyan``` | color
```-``` | darker color
